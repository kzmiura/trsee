## What is

`tree` but in Rust lang.

## How to

```bash
trsee -h
```

```
Usage: trsee [OPTIONS] <DIR>

Arguments:
  <DIR>  

Options:
  -d, --depth <DEPTH>    
  -a, --show-hidden      
  -s, --show-summary     
  -f, --follow-symlinks  
  -h, --help             Print help
  -V, --version          Print version
```
